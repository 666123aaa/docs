02 — Game / interactive media

Interactive Extension

The interactive work translates the video’s critique into procedural experience: prompts and “choices” operate as micro-instruments of discipline, where agency is staged as a managed interface rather than freedom. The player’s actions become a trace of how subjectivity is formatted under platform logic.

I produced a dedicated web version by porting/transcribing the original Processing game so it can run, be accessed, and be evaluated in a browser-based portfolio context.
